Numeric Types +

  • smallint: Stores small integers (-32,768 to 32,767).
  • integer: Stores standard integers (-2,147,483,648 to 2,147,483,647).
  • bigint: Stores large integers (-9 quintillion to 9 quintillion).
  • decimal/numeric: Exact numeric type with precision.
  • real: 4-byte floating-point number.
  • double precision: 8-byte floating-point number.
  • serial: Auto-incrementing integer.
  • bigserial: Auto-incrementing big integer.

Character Types +

  • char(n): Fixed-length character string.
  • varchar(n): Variable-length character string.
  • text: Variable-length character string (unlimited).

Boolean Type +

  • boolean: Stores `TRUE`, `FALSE`, or `NULL`.

Date/Time Types +

  • date: Stores date (year, month, day).
  • timestamp: Stores date and time.
  • interval: Stores a time span.

UUID Type +

  • uuid: Stores universally unique identifiers.
